# Аниматоры

Аниматоры предназначены для высокоуровневого управления анимациями: Состояния, Смешивания, Слои.

> [!NOTE]
> На данный момент поддерживается только управление состояниями

Аниматоры декларируются в **JSON** файлах в следующем формате:

## Свойства

- `animations` - таблица, где ключи - пути к файлами с анимациями,
значения - имена анимаций, которые нужно использовать в аниматоре;

> [!WARNING]
> При загрузке аниматора будет ошибка, если используются одноимённые
> анимации из разных файлов.

- `states` - массив имён полей состояний;

- `conditions` - таблица с условиями активаций различных анимаций,
где ключи - имена анимаций, значения - **Lua** выражения, использующие
объявленные в аниматоре поля для контроля состояний, и возвращающее
булевое значение - `true`/`false` определяющие, должна ли
включиться анимация.

**Пример:**
```json
{
    "animations": {
          "coolmod:animations/base.lfa": [ "walking", "running", "idle" ],
          "coolmod:animations/aggresive.lfa": [ "attacking" ]
    },

    "states": [ "walk", "run", "attack" ],
  
    "conditions": {
          "walking": "walk and not run",
          "running": "walk and run",
          "attacking": "attack"
    }
}
```
