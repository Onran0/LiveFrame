# Введение в LiveFrame API

Для начала вам нужно ознакомиться с форматами анимаций, поддерживаемыми **LiveFrame**:
- **LFA** (LiveFrame Animation) - первый и самый основной, родной формат **LiveFrame**.
Документация о формате [здесь](lfa.md).

Далее, вам нужно определиться, какой именно способ анимирования вам нужен:
- Простой проигрыватель (`player`), если вам нужно проигрывать одну или несколько анимаций
без переходов и смешиваний;
- Аниматор (`animator`), если вам нужны переходы, слои и смешивания.

Если вы выбрали проигрыватель:
- С помощью [liveframe:player](scripting/components/player.md) вы можете легко добавить компонент
проигрывателя на вашу сущность.

Если вы выбрали аниматор:
- С помощью [liveframe:animator](scripting/components/animator.md) вы можете легко добавить компонент
аниматора на вашу сущность;
- Также, вам необходимо прочесть документацию об [описании аниматоров](animators.md).

А документация о модуле [liveframe:api](scripting/modules/api.md) и классах [player](scripting/classes/player.md),
[animator](scripting/classes/animator.md), [sampler](scripting/classes/sampler.md)
поможет вам ознакомиться с написанием обработчиков ивентов,
или разработкой более сложных сценариев использования.
