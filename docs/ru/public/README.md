# Введение в LiveFrame API

Для начала вам нужно определиться, что именно вам нужно:
- Простой проигрыватель (`player`), если вам нужно проигрывать одну анимацию без переходов и смешиваний.
- Аниматор (`animator`), если вам нужны переходы, слои и смешивания.

Если вы выбрали проигрыватель:
- С помощью [liveframe:player](scripting/components/player.md) вы можете легко добавить компонент
проигрывателя на вашу сущность.

Если вы выбрали аниматор:
- С помощью [liveframe:animator](scripting/components/animator.md) вы можете легко добавить компонент
аниматора на вашу сущность.

А документация о модуле [liveframe:api](scripting/modules/api.md) и классах [player](scripting/classes/player.md),
[animator](scripting/classes/animator.md) поможет вам ознакомиться с написанием обработчиков ивентов,
или с более сложными сценариями использования.
