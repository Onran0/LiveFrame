# Объект clip

**Описание**

Используется для хранения необходимых и оптимизированных данных для проигрывания клипа.

**Структура**

> [!NOTE]
> `*` слева от названия свойства означает, что оно опционально.

> [!NOTE]
> Зависим от конкретного [clips_metadata](clips_metadata.md)!

```lua
clip = {
    -- имя клипа
    name: string ,

    -- зациклен ли клип
    loop: boolean,

    -- длительность клипа
    duration: number,

    -- массив с индексами костей, у которых в клипе есть хотя бы одна трансформация
    affectedBones: table<int>,

    -- массив с событиями клипа
    events: table = {
        {
            -- имя ивента
            name: string,
                
            -- время ивента в секундах
            time: number,
                
            -- значение ивента
            *value: any
        },

        ...
    },

    -- массив с ключ-кадрами для компонентов позы костей.
    -- без пропусков! если трансформаций нет, задавайте таблицы { { }, { }, { } }.
    bonesKeys: table {
        [индекс_кости]: table = {
            -- ключ-кадры для позиции
            [1]: table = {
                {
                    -- значение позиции
                    [1]: table = { x: number, y: number, z: number  }
    
                    -- время ключ-кадра в секундах
                    [2]: number,
    
                    -- индекс типа интерполяции, используемый в переходе к следующему ключ-кадру позиции.
                    -- может быть не указан, если ключ-кадр последний и клип не зацикленный.
                    -- в ином случае, обязателен.
                    *[3]: int,
    
                    -- значения для полей интерполяции.
                    -- обязателен и имеет смысл в определении при типах интерполяции, требующих
                    -- больше вводных данных, чем a; b; t, например cubic-spline или squad
                    *[4]: table = {
                        [индекс_поля_типа_интерполяции]: any,
                        ...
                    }
                },
                
                ...
            },
    
            -- ключ-кадры для вращения
            [2]: table = {
                {
                -- значение вращения
                [1]: table = { w: number, x: number, y: number, z: number  }
                
                -- то же, что и в [1]
                },
            }
    
            -- ключ-кадры для масштаба
            [3]: table = {
                -- значение позиции
                [1]: table = { x: number, y: number, z: number  }
                
                -- то же, что и в [1]
            }
        },

        ...
    }
}
```
