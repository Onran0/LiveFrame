# Класс sampler

**Описание**

Используется для вычисления сэмплов трансформаций конкретной кости или всех на основе метаданных и некоторых аргументов.

**Используемые определения:**

- `метаданные клипов` - объект, содержащий информацию о ключ-кадрах
клипов и доп. информацию (см. [clips_metadata](../objects/clips_metadata.md));

**Используемые классы & объекты:**
- [clips_metadata](../objects/clips_metadata.md);
- [clip](../objects/clip.md). 

```lua
--- Сэмплинг ---

-- Возвращает индекс кости в метаданных клипов по её имени
sampler:get_bone_index_in_meta(boneName: string) -> int

-- Позволяет получить сэмпл трансформаций конкретной кости.
-- Возвращает позицию, вращение и масштаб.
sampler:get_bone_transform_sample(
        -- Индекс кости в метаданных клипов
        boneIndex: int,
        
        -- Время сэмпла (не нормализованное, должно быть в диапазоне [0; длительность_клипа])
        currentTime: number,
        
        -- Индекс клипа, из которого будет вычисляться сэмпл
        clipIndex: int,
        
        -- Аргумент, с помощью которого можно обернуть результат в единую таблицу
        [опционально] returnTable: boolean = false
) -> (vec3, quat, vec3) | { vec3, quat, vec3 }

-- Позволяет получить все сэмплы трансформаций всех костей
-- В результате:
-- ключ - индекс кости в метаданных клипов;
-- значение - трансформации кости в таблице (позиция, поворот, масштаб).
sampler:get_transforms_sample(
        -- Время сэмпла (не нормализованное, должно быть в диапазоне [0; длительность_клипа])
        currentTime: number,
        
        -- Индекс клипа, из которого будет вычисляться сэмпл
        clipIndex: int
) -> table<int={ vec3, quat, vec3 }>

--- Работа с информацией клипов ---

-- Получить индекс клипа по имени
sampler:get_clip_index_by_name(name: string) -> int

-- Получить имя клипа по индексу
sampler:get_clip_name_by_index(index: int) -> string

-- Получить объект клипа по индексу
sampler:get_clip_by_index(index: int) -> clip

-- Получить объект клипа по имени
sampler:get_clip_by_name(name: string) -> clip

-- Возвращает true, если клип с указанным индексом зациклен
sampler:is_clip_looped(index: int) -> boolean

-- Возвращает длительность клипа по индексу
sampler:get_clip_duration(index: int) -> number

-- Возвращает метаданные клипов, используемые сэмплером
sampler:get_clips_metadata() -> clips_metadata
```
